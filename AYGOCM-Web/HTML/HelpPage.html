1. System Requirements

Please ensure that you definitely match the minimal system requirements as shown below:

  • Display: a minimal resolution of 1024 x 768
  • Operation System: Windows Vista/7/8/8.1/10

    • (Windows XP not supported.) [Click here to see why][Click here to hide]
    Since AYGOCM requires your computer to have .NET Framework 4.5 installed, the program will not work in Windows XP as the latest .NET that XP supports is 4.0. We have a Java Edition available here!

  • Processor: 1.6GHz processor

    • (Optimized for both single and multicore processors)

  • Hard Disk Space: 50MB Free hard drive space (Installer files)
  • RAM: 1024MB (1GB)

    • Others/Prerequisites:

  • Microsoft .NET Framework 4.5
  • Internet connection (For update checking, news and FAQ)
  • Mouse and Keyboard

2. Downloading and Installing the program
  • Download the program from the website below:

    Anime Yu-Gi-Oh! Card Maker Download Page

  • Alternatively, click here for a direct download link: EXE | JAVA | Android (unavailable)
  • If you have downloaded the Java version then do the following:
    1. Make sure JRE 8+ is installed, then simply launch with a double click.
    2. Alternatively, you can run the command java -jar AnimeYuGiOhCardMaker.jar in a command line interface.
  • If you have downloaded the EXE version then do the following:
    1. Make sure .NET Framework 4.5 is installed.
    2. Simply launch the program by double clicking it.
  • If successful, you'll now have the application running on your computer, if something went wrong, please provide a screenshot and send it to: aygocm@gmail.com

3. How to use the program
  • The Interface


    1. ATK Field - This is the attack value of a card (usually in multiples of 50> or 100. E.g. 1500)
    2. DEF Field - This is the defense value of a card (usually in multiples of 50> or 100. E.g. 1500)
    3. Level Bar - This shows how many levels a Monster Card has. This can be modified using buttons shown in #5
    4. Attribute Icon - This shows the attribute of a Monster Card. This can be modified using buttons shown in #6
    5. Level Types & Numbers - There are 3 different Level types:
      Normal, Rank, Negative.
    6. Attribute Types - There are 7 types of attributes to choose from, these will be displayed on the card as shown by #4
      Earth, Water, Fire, Light,
      Divine, Dark, Wind.
    7. Card Templates - These images change the appearance of the Card on the left. This change happens immediately when clicking on each image.
      When you hover your mouse over a template for a few seconds, a popup message will tell you the name of each template.
    8. Adding Images - Images can be added to the big white squared space where the + symbol is located. The + symbol can be clicked, or the button can be pressed (located to the right).
      Both actions will open a Dialog where you can choose an image (of type .PNG, .JP(E)G, .BMP) which will then be displayed int he white space of the card.
    9. File Name Text Box - This will be the name of your card once you decide to save it. You can write anything in here. (e.g.
    10. File Type Drop Box - In relation to #9, This is the Image format type that your card will be saved as.
      Usually, this box can be ignored as in most cases the value does not need to be changed.
    11. Card Library - This button (located bottom-right of the big card) opens another window that shows all your created cards, from there they can be viewed, or deleted.
      The location of this can be changed under the settings tab at the top of the app.
    12. Reset Card - This is the button near the bottom-right of the app, with the bright red text. It resets any changes or modifications you have made. Essentially, the app will look exactly like it does in the image above.
    13. Help Guide - The button opens this Dialog!
    14. Check for Updates - This checks the server for any new updates of the card maker.
    15. Report Bug - Did you come across something that wasn't working as intended? Please press this button and submit your report here. You may also include images. Please try to be as detailed as possible.
    16. Feedback Button - Can you think of something that could make this app even better? Please submit it by pressing this button! It will take you to a Google Forms page.
    17. Export Card Button - Once you have designed the card to your choosing, pressing this button (the really wide button towards the middle) will save your card to the library location mentioned in #11
    18. Save/Load Configuration - These buttons allow you to save a card configuration for if you ever want to recreate a card with ease. The editor allows you to change values for a save file. This is shown in the Save Editor section
    19. Extras Group - This section contains 4 buttons:
      - Shows a menu where you can put extra symbols into ATK/DEF fields (#1, #2) such as the infinite symbol.
      - This opens a console that allows automation of card images, this uses the Search Tool function described below.
      - This search tool function allows you to search for a specific card on the YGOPRODeck Database and apply it to the card in the app.
      - This button opens a dialog where you can pick a YGOPRO deck file (.ydk) and import all the cards into the app to convert them into anime versions.
    20. Login/My Account Button - This button brings you tot he account section of the app, this is where you can register an account, and redeem your Testers value. Accounts in the future will allow you to share your creations with the world.

Using the Save Editor

This tutorial will show you how to use the Save Editor.

  • First, press the button on the program. (Located next to the and buttons)
  • Next, choose a saved card configuration and open it up with the file explorer
  • If the file loaded successfully, then all the empty spaces should get filled in, if this isn't the case, please refer to the guide.
  • Below you will find the values that can go in each field:

Levels:
  • All level types can hold up to a value between 1 and 13.
  • For normal levels, you can use value 14 to get a ? level.

NoMonster:
  • NoMonster has 2 values, 0 and 1. 0 meaning False, and 1 meaning True.
  • Value 0 indicates that the card is a monster.
  • Value 1 indicates that the card is not a monster.

Spell Card:
  • Spell Card has value True and False.
  • If set to True, the card will be seen as a Spell Card.
  • If set to False, the card will not be seen as a Spell Card.

Trap Card:
  • Trap Card has value True and False.
  • If set to True, the card will be seen as a Trap Card.
  • If set to False, the card will not be seen as a Trap Card.

XYZ Monster:
  • (This setting should only be used if you are using version 4.1 or below, however version 5.0+ will still be able to read this setting from older configurations)
  • XYZ Monster has 2 values, 0 and 1. 0 meaning False, and 1 meaning True.
  • Value 0 indicates that the card should use XYZ levels.
  • Value 1 indicates that the card should not use XYZ levels.

Level Type:
  • (This setting only applies to version 5.0 or newer and replaces "XYZ Monster" setting)
  • Level Type has 3 values: N, X and D.
  • Value N indicates that the card should use Normal levels.
  • Value X indicates that the card should use XYZ levels.
  • Value D indicates that the card should use Negative/Dark levels.

ATK/DEF Value:
  • These two fields specify the ATK/DEF points that the card should have.
  • Although you can type values that have more than 5 characters, the program will only display a maximum of 5 characters.

Attribute ID:
  • This field specifies the attribute that the card will have, by calling out its ID.
  • ID's go from 1 to 7.
  • Here are the following ID's for each attribute:
  • Earth - 1
  • Water - 2
  • Fire - 3
  • Wind - 4
  • Dark - 5
  • Light - 6
  • Divine - 7
  • Typing any other value will not be read by the program.

Card ID:
  • This field specifies the template that the card will use, by calling out its ID.
  • ID's go from 1 to 21.
  • The list is huge so we put it into a Spoiler:

    • Fusion - 4

    • Slifer the Sky Dragon - 9
    • Spell - 10
    • Trap - 11
    • Legendary - 12
    • Normal 1-Box - 13
    • Effect 1-Box - 14
    • Alternative Ritual - 15

    • Effect Pendulum - 21
    • ATK/DEF Token - 22
    • Dark Synchro - 23
    • Custom Normal - 24
    • Ritual Pendulum - 25
    • Fusion Pendulum - 26
    • Synchro Pendulum - 27
    • XYZ Pendulum - 28
    • Dark Synchro Pendulum - 29
    • LINK Monster - 30
    • Typing any other value will not be read by the program. Using a custom template will reset the Card ID back to 1

Custom Card:
  • Custom Card has value True and False.
  • If set to True, the card will be seen as a Custom Card.
  • If set to False, the card will not be seen as a Custom Card.
  • If Custom Card is True then the value for Card ID is irrelevant.
